HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The numbered h elements represent the level of heading a given text box represents.
  • The paragraph element represents a paragraph or block of text.
  • The list elements represent undordered lists, ordered lists, and list items.
  • The image element contains information about the images contained on a webpage.
  • The break element creates a line of empty space between two blocks of content.
  • The anchor element creates a link to the page or another website.
  • The header element is usually at the top of the page and contains introductory content.
  • The footer element is usaully at the bottom of the page and contains author, copyright, sitemap, and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A box-shadow adds shadow effects around an element's frame.
  • Assigning a color can be done by using semantic references or using a hexadecimal code.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Contol flow is the order in which a computer executes code in a script.
